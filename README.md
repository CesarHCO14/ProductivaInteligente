# Productiva Inteligente
Proyecto Android usando base de datos local relacional en 3FN mediante Room sobre SQLite.

## Características
- Login local
- Dashboard
- Gastos
- Hábitos con seguimiento diario
- Recordatorios
- Informes con gráficos
- Base relacional normalizada
- Funciona sin backend ni internet

## Cómo abrir
1. Descomprime `android-app`
2. Abre Android Studio
3. File > Open > selecciona `android-app`
4. Usa Embedded JDK
5. Espera Sync
6. Ejecuta
